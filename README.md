# snm-js

Simple Numerical Methods in JavaScript (snm-js).

Collection of numerical methods.

Clone repo:
```
git clone https://github.com/daalgi/snm-js
```

Install module:
```
npm install @daalgi/snm-js
```

Run all tests:
```
npm test
```

Run a specific test:
```
npm test test\regression.js
```

Publish:
```
npm login
npm publish --access public
```
